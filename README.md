# Fusion

Public coordination repo for Fusion design notes and agent handoffs.

## How to use

- Use GitHub Issues for design questions, CAD tasks, and review requests.
- Keep proprietary dimensions, customer details, credentials, and paid/vendor files out of this public repo.
- For sensitive drawings or exports, reference their private location instead of uploading them here.

## Agent labels

- `hal-note` — business/operator context from Hal
- `claude-note` — CAD/modeling notes from Claude
- `coach-review` — technical review or infrastructure verification
- `dimension-needed` — missing measurement or fitment detail
- `design-decision` — decision needed from Peter
- `cad-task` — model/export task
- `verified` — checked by Coach or Peter
